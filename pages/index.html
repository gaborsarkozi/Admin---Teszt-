Megújult áruházunkhoz újragondolt admin felületet hoztunk létre.

Az áruház admin oldala funkcióiban biztosítja a régi admin oldal lehetőségeit, megjelenésében pedig igazodik az új áruház látványelemeihez. A már meglévő funkciók mellé új elemek kerültek a rendszerben, valamint folyamatosan fejlesztjük a rendszert, hatékonyságát növeljük az optimális működés érdekében. Amennyiben az admin oldal kezelőfelületének használatában problémák merülnek fel, az admin oldal útmutatóban található leírások segítenek megkönnyíteni a működést.
