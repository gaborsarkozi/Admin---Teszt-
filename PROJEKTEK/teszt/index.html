Gradient generator

Véletlenszerű gradient átmenetek generálása, amivel teljesen egyedi színkeveréket kapsz. A kódmásoló segítségével a a kapott színhez tartozó rögtön beilleszhető a CSS file-odba.

background: #c32bad;
background: -moz-linear-gradient(256deg, #c32bad 0%, #99feff 100%);
background: -webkit-linear-gradient(256deg, #c32bad 0%, #99feff 100%);
background: linear-gradient(256deg, #c32bad 0%, #99feff 100%);
